> This is the official repository for the paper **"RailVQA: A Benchmark and Framework for Efficient Interpretable Visual Cognition in Automatic Train Operation"**[cite: 2].

Autonomous Train Operation (ATO) requires highly reliable cab-view visual perception and decision-oriented inference[cite: 5]. To address the limitations of existing perception-only methods and the high computational cost of Large Multi-modal Models (LMMs), we introduce **RailVQA-bench**, the first comprehensive VQA benchmark for ATO, and **RailVQA-CoM**, a collaborative large-small model framework. For our whole work, please check in https://cybereye-bjtu.github.io/demo.html#features.
